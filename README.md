# 🛒 MERN E-commerce Application

A full-stack eCommerce web application built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**.  
This application simulates a real-world online shopping system where users can browse products, place orders, and track their order status, while admins manage products, users, and orders through a dedicated dashboard.

## ✨ Features

### 👤 User Features
- User Registration & Login (JWT Authentication)
- Secure Logout System
- Browse Products
- Select Products
- Add to Cart
- Remove from Cart
- Proceed to Checkout
- Place Orders
- View Order Status:
  - Pending (waiting for admin approval)
  - Approved
  - Delivered
- View Personal Orders

---

### 🛠️ Admin Features
- Admin Login / Logout
- Create Products
- Update Products
- Delete Products
- View All Orders
- Approve Orders (Pending → Approved → Delivered)
- View All Users
- Delete Users
- View Total Revenue
